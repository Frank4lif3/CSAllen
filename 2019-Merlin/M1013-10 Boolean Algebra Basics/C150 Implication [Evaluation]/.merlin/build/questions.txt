2.  ¬true → false 
3.  ¬true ↔ ¬false 
4.  true → ¬false 
5.  ¬true ↔ ¬true 
6. ( false ⊽ ¬true ⊼ ¬true ) ← ( ¬false ⊻ ¬true ⩒ ¬true )
7. ( ¬true ∨ true ⊼ false ) → ( ¬true ⊼ ¬false ⊽ false )
8. ( false ⊼ ¬false ⊼ true ) ← ( true ⊻ true ∧ true )
9. ( false ∧ ¬false ∧ true ) ↔ ( true ⊻ false ⊕ false )
10. ( false ∧ ¬true ⊻ false ) ↔ ( ¬true ⊼ false ⊻ true )
